##  信息素养的含义

**信息素养是一种信息文化，它是一种基本能力，与基本学习技能（读、写、算）、人际交往和合作精神、创新思维能力、实践能力一起构成人的面向21世纪的能力素养；另外它是一种综合能力，它涉及各方面的知识，是一个特殊的、涵盖面很宽的能力。信息素养是一种信息能力，信息技术是它的工具。**
